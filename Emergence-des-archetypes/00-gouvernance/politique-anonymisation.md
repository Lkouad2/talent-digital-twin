# Politique d’anonymisation

## Périmètre public

Le dépôt est public. Toute information publiée doit être considérée comme accessible, indexable et durable.

## Données supprimées

- nom et prénom ;
- pseudonyme public utilisé sur un forum ;
- adresse électronique ou téléphone ;
- entreprise ou équipe identifiable ;
- ville précise ;
- âge exact ;
- date d’entretien ;
- enregistrement audio ou vidéo ;
- verbatim sensible ;
- URL individuelle permettant une ré-identification immédiate.

## Données généralisées

| Donnée brute | Donnée publique |
|---|---|
| 48 ans | 45–54 ans |
| Directeur dans une entreprise nommée | Fonction de direction dans une organisation |
| Métier très rare + secteur + ville | Famille de rôle généralisée |
| Citation exacte | Paraphrase analytique |
| URL d’un témoignage individuel | Type de source ou registre privé |

## Pseudonymisation

Les personnes sont identifiées par `ENT-XXX`. Aucune clé de correspondance n’est publiée dans le dépôt.

## Minimisation

Une information n’est publiée que si elle est nécessaire pour :

- comprendre le problème ;
- justifier un code ;
- soutenir ou contredire une hypothèse ;
- expliquer une décision.

## Contrôle avant publication

- [ ] Aucun nom de personne
- [ ] Aucun employeur identifiable
- [ ] Aucun verbatim sensible
- [ ] Aucun enregistrement brut
- [ ] Aucun lien individuel ré-identifiant
- [ ] Aucun fichier privé inclus par erreur
