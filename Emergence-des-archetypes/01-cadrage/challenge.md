# Challenge

## Problème central

Les professionnels expérimentés ne manquent pas seulement d’offres d’emploi.

Ils manquent d’une représentation continue, fidèle et personnalisée de leur potentiel professionnel.

Les outils actuels décrivent principalement :

- l’intitulé du poste ;
- les diplômes ;
- les certifications ;
- les mots-clés du CV ;
- les expériences faciles à raconter.

Ils rendent difficilement visibles :

- les apprentissages réalisés dans les projets ;
- les compétences acquises entre pairs ;
- la progression de la complexité des responsabilités ;
- les compétences exercées à l’interface de plusieurs métiers ;
- l’impact produit sur les équipes ;
- les capacités transférables vers d’autres fonctions.

## Premier risque à tester

> Le service est-il capable de comprendre une personne mieux que son CV, et la personne se reconnaît-elle dans cette représentation ?

## Critères de succès

- reconnaissance de la majorité des propositions ;
- preuves compréhensibles ;
- découverte d’au moins une valeur utile ;
- possibilité de corriger et rejeter ;
- confiance suffisante pour réutiliser le profil.
