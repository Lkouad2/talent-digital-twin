# Sources secondaires

Les sources secondaires servent à :

- tester la plausibilité des hypothèses ;
- préciser les mécanismes ;
- identifier des profils extrêmes ;
- préparer les questions de terrain.

Les identités des personnes citées dans des entretiens publics ne sont pas reproduites dans le dépôt.

Le registre exact des URLs et correspondances reste privé afin d’éviter la ré-identification par combinaison d’indices.
