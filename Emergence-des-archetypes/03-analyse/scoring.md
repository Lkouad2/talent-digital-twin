# Scoring

## Score d’une preuve

```text
score brut =
directivité × qualité × pertinence cible × indépendance
```

## Directivité

| Valeur | Interprétation |
|---:|---|
| 0 | information absente |
| 1 | indice indirect |
| 2 | preuve partielle |
| 3 | preuve directe |

## Polarité

| Polarité | Effet |
|---|---:|
| Soutient | score positif |
| Nuance | 50 % |
| Contredit | score négatif |
| Neutre | 0 |

## Agrégation

Une personne ou une étude ne compte qu’une fois par mécanisme :

- preuve directe : facteur 1 ;
- preuve partielle : facteur 0,5.

## Décision

Un mécanisme est central provisoire lorsqu’il présente :

- un score final élevé ;
- plusieurs sources distinctes ;
- des entretiens proches de la cible ;
- au moins une source secondaire indépendante ;
- des contradictions documentées.
