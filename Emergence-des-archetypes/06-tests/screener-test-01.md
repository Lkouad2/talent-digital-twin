# Screener TEST-01

## Critères

- 35 à 55 ans ;
- dix ans d’expérience ou trajectoire riche ;
- au moins une transition technologique, fonctionnelle ou organisationnelle ;
- deux projets précis à raconter ;
- disponibilité pour un test de 60 minutes.

## Questions de mécanisme

- R1 : avez-vous participé à une migration ou appris une technologie pour un projet ?
- R2 : l’échelle de vos projets a-t-elle fortement augmenté ?
- R3 : travaillez-vous entre plusieurs univers qui se comprennent mal ?
- R4 : des collègues viennent-ils vers vous pour apprendre ou décider ?

## Quotas

- 3 profils R1 ;
- 3 profils R2 ;
- 3 profils R3 ;
- 3 profils R4 ;
- 3 profils poly-R ;
- 2 contre-cas.
