# Dimensions du potentiel

Les quatre R sont des dimensions combinables.

| Dimension | Question centrale |
|---|---|
| R1 | Quelles couches technologiques et quels principes ont été accumulés ? |
| R2 | Comment l’échelle et la complexité des projets ont-elles progressé ? |
| R3 | Quelles frontières la personne relie-t-elle ? |
| R4 | Qu’est-ce que les autres savent faire grâce à elle ? |
